
#### Project 2: "Check out the music : Hypothesis about streaming service"

##### Description:
 Data analysts extract valuable insights from data to make informed decisions, involving data overview, preprocessing, and hypothesis testing. In this project, we will explore the differences in music preferences between residents of Springfield and Shelbyville using data from an online music streaming service. Test the hypothesis that "user activity differs depending on the day of the week and the city". The analysis aims to identify distinct patterns in music preferences between Springfield and Shelbyville users and how these patterns vary throughout the week.
##### Key Technologies:
Python, Pandas, etc.
##### Key Skills: 
Data preprocessing, exploratory data analysis, statistical analysis, etc.
##### Language:
Portuguese
